# Tinypokemon

Java Programming Class Final: Small imitation of Pokemon game

#### Requirements
* Pokemon base class
* SuperPokemon class (Extends Pokemon base class and has an extra Super Ability)
* Constructor
* Setters and getters for a Pokemon's
  * Name
  * Health points
  * Energy
  * Unique Ability
* Print Pokemon's information
* Imitate a battle
